رفع المشروع على GitHub Pages

اتبع الخطوات التالية لرفع مشروعك ومشاركة الرابط مع الدكتور

1

حمّل الملف على جهازك

تأكد أن ملف index.html محفوظ في مجلد باسم مخزني على سطح المكتب

2

افتح GitHub وأنشئ Repository جديد

اذهب إلى github.com → اضغط New Repository → سمّه makhzani → اختر Public → اضغط Create

3

ارفع الملف مباشرة من الموقع

في صفحة الـ Repository اضغط uploading an existing file ثم اسحب ملف index.html وأفلته، ثم اضغط Commit changes

4

فعّل GitHub Pages

اذهب إلى Settings → Pages → في Source اختر main → اضغط Save

5

احصل على الرابط 🎉

بعد دقيقتين سيظهر رابطك:

https://اسم-حسابك.github.io/makhzani

أرسل هذا الرابط للدكتور وسيفتح المشروع مباشرة! ✅
